# 11x AI SDR Integration

**Type**: AI SDR agent (Alice / Mike) — autonomous email + LinkedIn outreach
**Best for**: Scaling outbound without adding SDR headcount; auto-personalized cadences

---

## Push Patterns

- Action: `clay/11x-push`
- Required: `agent_id`, `lead_data`
- 11x then runs autonomous outreach (sending, replies, follow-ups)

## Mandatory: Sending Gate Filter

```
Action: push-11x
Run Condition: sending_gate_eligible = TRUE
```

## Critical: Coordinate with 11x's Own ICP Filter

11x agents have their own ICP filter inside the platform. Coordination rule:

```
Clay sending_gate_eligible → STRICTER filter (you trust Clay's gate)
11x ICP filter → LOOSER or OFF

Otherwise: double filtering silently drops eligible rows.
```

Always: configure 11x to accept what Clay pushes. Move the filter logic into Clay where you control it.

## Personalization Handoff

11x can do its own personalization, but it's expensive and slower. Better pattern:
1. Clay does Claygent research → produces `research_summary`, `pain_signal`, `first_line`
2. Clay pushes these to 11x as input fields
3. 11x weaves them into its sequencer using its own template logic

This way Clay credits do the research once; 11x just executes the cadence.

## When to Use

- ✅ Scaling outbound beyond SDR headcount capacity
- ✅ Specific high-volume motion (job change reach-outs, signal-driven sprints)
- ✅ Mid-market SaaS where AI tone is acceptable

## When NOT to Use

- ❌ Enterprise / high-touch sales (human SDR preferred)
- ❌ Regulated industries (Bank, Insurance, Healthcare) — compliance risk on AI-generated copy
- ❌ Brand-sensitive companies where founder voice matters

## Common Issues

| Symptom | Cause | Fix |
|---------|-------|-----|
| Leads pushed but no emails sending | 11x ICP filter rejecting silently | Loosen 11x filter, tighten Clay gate |
| AI tone off-brand | 11x default voice not customized | Provide voice samples + brand guidelines to 11x setup |
| Replies not handled | Webhook from 11x → Clay not configured | Set up reply → Clay reply tracking column |

## Notes

- 11x's value scales with volume. Sub-500 contacts/week → manual SDR or SalesLoft is cheaper.
- Always verify the first 50 messages 11x sends before going to full volume — AI drift happens.
